# POWER SCALE 管理範本

戰力與能力系統是長篇作品易失衡的部分。為防止戰力膨脹與無代價升級，請在 `06_PowerScale/` 目錄中建立每個角色或種族的戰力記錄，並定期更新。以下範本可用於撰寫相關檔案。

```markdown
## 名稱（角色 / 種族 / 陣營）

- **目前戰力等級**：描述當前階段的戰力等級，可使用具體數值或等級系統。
- **能力描述**：概述能力或武器的特性、優勢與使用方式。
- **限制與代價**：列出此能力需要付出的代價或限制，例如能量消耗、傷害反噬、時間間隔等。
- **弱點**：描述易被克制的方式或自然弱點。
- **可擊敗對象**：現階段可以戰勝的對手或威脅。
- **不可擊敗對象**：現階段難以或無法戰勝的對手。
- **下一階段成長條件**：升級或進化需要滿足的條件，如完成修煉、獲得某道具或解開封印等。
- **是否成長過快**：檢查戰力增長速度是否合理，並提醒避免無代價升級。
```

**注意**：

- 更新戰力檔案時必須同步更新章節描述與角色資料卡，避免設定不一致。
- 如果在 QA 階段發現戰力失衡，需要回滾或調整章節內容。
