# CHARACTER 資料卡範本

每個重要角色必須建立一個獨立的 Markdown 檔案，存放於 `03_Characters/` 目錄。請使用以下範本撰寫角色資料卡，保持欄位完整與格式一致，便於查找與機器讀取。

```markdown
## 角色名稱

- **定位／角色類型**：主角 / 配角 / 反派 / 群像之一 / NPC 等。
- **初始性格**：簡要描述角色的性格特質，例如內向、自信、狡猾、勇敢等。
- **核心慾望**：角色最渴望達成的目標或願望。
- **恐懼與弱點**：角色害怕的事物或內心弱點。
- **能力與技能**：角色擁有的特殊能力、戰鬥技能或知識專長，並說明其限制與代價。
- **成長線／進化路線**：分階段描述角色在故事中的能力或性格成長，例如初始 → 中期 → 最終形態。
- **關係網**：列出角色與其他重要角色的關係，如敵對、盟友、家族、師徒等。
- **秘密與伏筆**：角色隱藏的秘密或將來會揭露的伏筆，需要在 Foreshadow Manager 中建立對應條目。
- **當前狀態**：角色在目前章節的狀態、情緒與地點。
- **不可違反規則**：列出此角色的核心設定與禁忌，例如「不使用某禁忌力量」等，這些不可隨意改動。
```

**注意**：

- 角色資料卡應在首次登場前建立，並隨劇情推進更新。
- 更新後必須於 `WORKLOG.md` 記錄修改與影響範圍。
