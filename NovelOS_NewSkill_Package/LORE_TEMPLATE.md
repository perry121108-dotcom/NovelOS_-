# LORE 資料範本

世界觀資料需存放於 `02_Lore/` 目錄下。可依類別拆分多個檔案，例如 `world_history.md`、`magic_system.md`、`religions.md` 等。建議按照下列欄位撰寫，以確保設定完整且條理清晰。

```markdown
## 世界觀主軸

- **背景概述**：簡述世界觀的整體背景與時代，例如：「一個科技與魔法並存的星球，經過漫長戰爭後分裂為五大陣營」。
- **種族與勢力**：列出各主要種族或族群，以及他們所在的勢力、國家或組織。
- **地理與地圖**：描述重要地理區域、城鎮或星系，最好搭配地圖文件（可附連結或附圖）。
- **歷史事件與傳說**：摘要影響當前局勢的重要歷史事件，並說明其後續影響。
- **宗教與神話**：列出影響世界觀的宗教體系、信仰與神話故事。
- **科技／魔法／能力規則**：詳細描述力量來源、運作原理、限制與代價。例如魔法需要消耗生命力、科技受到能量限制等。
- **進化與階級制度**：說明角色或種族的進化階梯及其社會階層，便於設計成長線與衝突。
- **重要物品與資源**：列出對劇情有關鍵影響的神器、資源或遺跡，並說明其特性與入手條件。
- **終局伏筆**：安置與主線終局相關的伏筆與神秘力量，並在 Foreshadow Manager 建立對應條目。
```

**注意**：

- 所有世界觀設定需經過 PM（ChatGPT）審核方可加入。
- 任何修改必須在 `WORKLOG.md` 記錄並於 `CHANGELOG.md` 更新版本號。
