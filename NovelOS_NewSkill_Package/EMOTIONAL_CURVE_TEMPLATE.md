# EMOTIONAL CURVE 範本

情緒曲線是長篇作品吸引讀者的重要元素。為設計適當的情緒起伏，請於 `07_EmotionalCurve/` 目錄建立每個篇章的情緒目標與節奏安排。建議使用下列欄位：

```markdown
## 篇章名稱 / 範圍

- **開端**：開篇或章節開頭的情緒狀態，通常用來引起注意或設置懸念。
- **中段**：鋪陳壓力或衝突的部分，持續累積緊張感。
- **高潮**：情緒爆發點，是本篇章最高潮，亦是劇情轉折的關鍵。
- **結尾**：高潮後的收束或留下懸念，引導讀者進入下一篇章。
- **情緒目標**：可從以下選擇或自定義：
  - 壓迫
  - 緊張
  - 爽感
  - 絕望
  - 希望
  - 悲傷
  - 成就感
  - 反轉
  - 餘韻
- **節奏提示**：提示段落轉換時的節奏，例如如何調整文字密度、對話比例或戰鬥描寫。
```

**注意**：

- 情緒曲線需與世界觀、角色成長與戰力設定一致。不能為了製造高潮而破壞邏輯。
- 若讀者反饋顯示節奏不佳，可於下一版本修正情緒曲線並更新 `CHANGELOG.md`。
