# FORESHADOW 管理範本

伏筆是長篇小說的重要構成。為避免遺漏或不合理回收，請在 `05_Foreshadow/` 目錄中使用此範本管理每個伏筆。每個伏筆應建立單獨的檔案（或集中於一個表格式檔案），並包含以下欄位。

```markdown
## 伏筆名稱

- **伏筆類型**：主線 / 世界觀 / 角色 / 情感 / 陰謀 / 終局 等。
- **首次出現章節**：指明伏筆最初出現的章節或篇幅。
- **涉及角色**：列出與此伏筆相關的角色。
- **內容描述**：簡要描述伏筆內容及其提示方式。
- **預計回收章節**：預估伏筆將於哪個章節或篇幅被揭露或回收，可修改但需記錄。
- **目前狀態**：未回收 / 部分回收 / 已回收。
- **回收方式**：簡述伏筆回收時的處理方式或劇情節點，可適時調整。
- **備註**：其他需要注意的事項，如與其他伏筆連動關係。
```

**注意**：

- 所有伏筆必須在合理章節回收，若預計回收時間變更，需更新此檔並記錄於 `WORKLOG.md`。
- 回收伏筆前，應確認相關的前期描寫、線索都已鋪陳完整。
